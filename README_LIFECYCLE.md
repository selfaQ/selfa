| المرحلة | الحالة | السلوك |

## 🚀 البدء السريع

### 1. تشغيل التطبيق
```bash
npm install  # إذا لم تفعل ذلك
npm run dev  # ابدأ الخادم
```

### 2. الذهاب إلى الصفحات
- **Browse:** http://localhost:5173/browse — شاهد المؤقت والمجموعات
- **Dashboard:** http://localhost:5173/dashboard — شاهد جمعياتك
- **مجموعة:** http://localhost:5173/salafa/1 — انضم لمجموعة

### 3. اختبر الميزات
- ✅ شاهد بانر المؤقت 24 ساعة
- ✅ انضم لمجموعة واختر مقعد
- ✅ شاهد شاشة النجاح
- ✅ تابع في لوحة التحكم

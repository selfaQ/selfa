# 📋 ميزة دورة حياة المجموعة (Group Lifecycle)

## نظرة عامة
صممت ميزة دورة حياة المجموعة لإدارة مراحل المجموعة الثلاث:

### المراحل الثلاث:

#### 🟢 **المرحلة 1 — مفتوحة (Open)**
- **التعريف:** المجموعة قيد الاستقبال وتجمع الأعضاء
- **السلوك:**
  - تظهر في صفحة **Browse** مع شريط تقدم يبين عدد الأعضاء الحالي
  - أول 3 مقاعد **مدفوعة** (أولوية مرتبة) — من يضغط أول يأخذ المقعد الأول
  - الأعضاء الباقون يدخلون **القرعة الحرة**
  
#### 🔵 **المرحلة 2 — مكتملة (Completed)**
- **التعريف:** المجموعة اكتملت عدد أعضائها (كل المقاعد ممتلئة)
- **السلوك:**
  - تختفي من صفحة **Browse**
  - تنتقل إلى **Dashboard** الشخصي لكل عضو
  - يظهر **مؤقت 24 ساعة** في أعلى صفحة Browse يعلم المستخدمين بموعد فتح مجموعة جديدة
  
#### 🟠 **المرحلة 3 — جديدة (New)**
- **التعريف:** بعد 24 ساعة من الاكتمال، تظهر مجموعة جديدة من نفس النوع
- **السلوك:**
  - المؤقت ينتهي
  - تظهر مجموعة جديدة من نفس المبلغ بعداد يبدأ من صفر

## التغييرات المطبقة

### 1️⃣ `src/data/constants.js`

#### إضافة حقول جديدة للمجموعات:
```javascript
{
  id: 1,
  amount: 5_000_000,
  duration: 10,
  filled: 7,
  priorityTaken: [true, true, false],
  startDate: '2025-05-01',
  status: 'open',              // ← جديد: حالة المجموعة
  completedAt: null,           // ← جديد: وقت الاكتمال (ISO format)
}
```

#### دوال مساعدة جديدة:

| الدالة | الوصف |
|-------|-------|
| `getTimerRemaining(completedAt)` | حساب الوقت المتبقي للمؤقت بالدقائق |
| `formatTimeRemaining(mins)` | تنسيق الوقت: "24 ساعة" / "12 س و 30 د" |
| `isTimeoutExpired(completedAt)` | التحقق إذا انتهت 24 ساعة |
| `filterByStatus(salafat, status)` | تصفية المجموعات حسب الحالة |
| `getDisplaySalafat(salafat)` | الحصول على المجموعات المفتوحة والجديدة (لـ Browse) |

### 2️⃣ `src/pages/Browse.jsx`

#### ✨ التحسينات:

1. **بانر المؤقت 24 ساعة**
   - يظهر في أعلى الصفحة عند وجود مجموعة مكتملة
   - يعرض الوقت المتبقي بشكل حي (يتحدث كل دقيقة)
   - يخبر المستخدم بموعد فتح مجموعة جديدة من نفس النوع

2. **التصفية الذكية**
   - تخفي المجموعات المكتملة من القائمة
   - تعرض فقط المجموعات المفتوحة والجديدة

3. **تحسينات عرضية**
   - إضافة عداد العضوية الحي
   - تحسين نصوص الحالة

```jsx
// مثال على بانر المؤقت:
{completedSalafa && timerMinutes > 0 && (
  <div className="bg-amber-50 border border-amber-200 rounded-xl p-4 mb-5">
    <p className="text-sm font-bold">⏱️ جمعية جديدة قادمة قريباً</p>
    <p className="text-lg font-black text-amber-600">
      {formatTimeRemaining(timerMinutes)}
    </p>
  </div>
)}
```

### 3️⃣ `src/pages/SalafaDetail.jsx`

#### ✨ التحسينات:

1. **شرح أنواع المقاعد**
   - قسم واضح يشرح الفرق بين المقاعد المدفوعة والقرعة
   - تحديد المزايا والفروقات

2. **نافذة التأكيد المحسّنة**
   - عرض نوع المقعد المختار (أولوية أم قرعة)
   - تفاصيل الدفع الكاملة:
     - القسط الشهري
     - رسوم الاشتراك
     - رسوم الأولوية (إن وجدت)
   - جدول الأقساط الكامل
   - **خطوات الانضمام الواضحة** (جديد)
   - تنبيه الخصم التلقائي

3. **شاشة النجاح المحسّنة**
   - عرض بصري جميل مع أيقونة وألوان
   - ملخص الانضمام والدور
   - قائمة بما سيحدث بعد الانضمام:
     - ✓ سيتم الخصم الشهري
     - ✓ ستستلم المبلغ كاملاً في شهرك
     - ✓ تابع التقدم من لوحتك
   - زر سهل للذهاب للوحة التحكم

```jsx
// مثال على شاشة النجاح:
<div className="bg-green-50 border-2 border-green-200 rounded-2xl p-5">
  <div className="flex items-center gap-2">
    <span className="w-5 h-5 rounded-full bg-green-500 text-white flex items-center justify-center">✓</span>
    <span>سيتم خصم {formatIQD(inst)} شهرياً</span>
  </div>
  {/* ... */}
</div>
```

### 4️⃣ `src/pages/Dashboard.jsx`

#### ✨ التحسينات:

1. **عرض حالة المجموعة**
   - عرض واضح إذا كان القسط مستلم أو قيد الانتظار
   - أيقون ✓ للأقساط المستلمة

2. **معلومات محسّنة عن الدور**
   - عرض "✓ استلمت" إذا تم الاستلام
   - عرض الشهر والسنة المتوقعة للاستقطاع

3. **شريط التقدم المحسّن**
   - عرض النسبة المئوية للتقدم
   - "X من Y قسط (X%)"

4. **معلومات الخصم التلقائي**
   - تظهر فقط إذا كان هناك قسط قادم
   - عرض "خصم تلقائي ✓"

```jsx
// مثال على بطاقة المجموعة:
<div className="grid grid-cols-3 gap-2">
  <div>
    <p className="text-[10px] text-slate-500">دوري</p>
    <p className="text-sm font-black text-primary">
      {monthsPaid >= myTurn ? '✓ استلمت' : `الشهر ${myTurn}`}
    </p>
  </div>
  {/* ... */}
</div>
```

## كيفية الاختبار

### 1️⃣ اختبار المؤقت (Browse)
```
✅ افتح صفحة Browse
✅ يجب أن ترى بانر "⏱️ جمعية جديدة قادمة"
✅ الوقت يتناقص كل دقيقة
✅ بعد 24 ساعة سيختفي
```

### 2️⃣ اختبار تفاصيل المقاعد (SalafaDetail)
```
✅ انقر على أي جمعية في Browse
✅ ستجد قسم "أنواع المقاعد"
✅ اختر مقعد أولوية أو قرعة
✅ في نافذة التأكيد ستظهر تفاصيل المقعد
```

### 3️⃣ اختبار شاشة النجاح (SalafaDetail)
```
✅ بعد تأكيد الانضمام
✅ ستظهر شاشة نجاح جميلة
✅ اضغط "اذهب إلى لوحتي"
```

### 4️⃣ اختبار لوحة التحكم (Dashboard)
```
✅ انضم لجمعية من Browse
✅ ستظهر في "جمعياتي"
✅ ستعرض حالة الدور والاستقطاع
```

# سلفة Q — ميزة دورة حياة المجموعة ✨

## 📌 الملخص

تم تطوير ميزة **دورة حياة المجموعة** مع 3 مراحل:

### المراحل:
1. **مفتوحة (Open)** 🟢 — تظهر في Browse مع شريط تقدم
2. **مكتملة (Completed)** 🔵 — مؤقت 24 ساعة + اختفاء من Browse
3. **جديدة (New)** 🟠 — مجموعة جديدة من نفس النوع بعد 24 ساعة

---

## 🎯 الصفحات المعدلة

### ✅ Browse.jsx
- **بانر مؤقت 24 ساعة** في الأعلى
- **إخفاء المجموعات المكتملة** من القائمة
- **تحديث حي** للمؤقت كل دقيقة
- **شريط تقدم واضح** لكل مجموعة

### ✅ SalafaDetail.jsx
- **شرح أنواع المقاعد** (أولوية vs قرعة)
- **نافذة تأكيد محسّنة** مع خطوات
- **شاشة نجاح جميلة** عند الانضمام

### ✅ Dashboard.jsx
- **عرض حالة المجموعة** الكاملة
- **شريط تقدم محسّن** مع نسبة مئوية
- **معلومات الدور والاستقطاع** واضحة
- **أيقونات حالة** (✓ مستلم / قيد الانتظار)

### ✅ constants.js
- **حقل status** للمجموعات (open/completed/new)
- **حقل completedAt** لتاريخ الاكتمال
- **دوال مساعدة جديدة** للمؤقت والحالات

## 🧪 كيفية الاختبار

### 1. اختبر المؤقت
```
1. افتح صفحة Browse
2. يجب أن ترى بانر "⏱️ جمعية جديدة قادمة"
3. الوقت يتناقص كل دقيقة
4. بعد 24 ساعة سيختفي
```

### 2. انضم لمجموعة
```
1. اضغط على أي مجموعة من Browse
2. اختر نوع مقعد (أولوية أو قرعة)
3. اضغط "تأكيد الانضمام"
4. ستظهر شاشة نجاح جميلة
```

### 3. تابع في Dashboard
```
1. اضغط "اذهب إلى لوحتي"
2. ستظهر المجموعة في "جمعياتي"
3. ستعرض دورك والاستقطاع القادم
```
